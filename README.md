## プラスコマンドについて
標準入力から読み込んだ数字の和を出力する.

## 使い方

```
$ seq 10 | ./plus
55
```

## 必要なソフトウェア
* Python
  *テスト済:ver.3.7～3.10

## テスト環境
* Ubuntu 20.04
